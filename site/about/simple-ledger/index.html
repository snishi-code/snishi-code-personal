家計簿

日々のお金の流れをもとに、今の生活を続けるためにどれだけお金が必要かを見直すための家計簿です。 支出、資産、負債、資金繰りを整理し、安心してお金を使える状態を目指します。

基本の使い方

ホーム画面の3ボタンから記録する
  • ホーム画面の 収入 / 支出 / 振替 から、日々の記録を追加できます。
  • お金の流れを「源泉 → 行き先」として選び、金額を入力します。
  • サブスク、年払い、耐久財などは「継続コスト化する」と台帳に登録できます。
  • 資産のバランス、収支のバランス、将来の資金繰りを確認できます。

使いこなすと便利なこと

生活会計を少ない手間で見直せます
  • 物の売却や処分を記録し、資産の残り方を整理できます。
  • カード払いやローンなどの負債と返済予定を確認できます。
  • 近い支払いに備えて、目的別に資金を取り分ける「取り置き資金」を管理できます。
  • JSON export/import でバックアップや端末移行ができます。

アプリを起動

家計簿アプリを開く
大切なデータは、定期的にJSONを書き出してバックアップしてください。
PWA・端末内保存 →

データとセキュリティ

入力データは端末内に保存されます
  • このアプリは外部サーバーへデータを送信しません。
  • JSON export/import は、バックアップと端末間移行のための機能です。
  • 端末の紛失、故障、ブラウザデータ削除に備えて、定期的にバックアップしてください。
  • 共有や同期が必要な場合は、アプリ外の手段でJSONファイルを管理してください。
